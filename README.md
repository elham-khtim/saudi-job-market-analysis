# Saudi Job Market Analysis Dashboard

This project presents a data-driven analysis of the Saudi job market, focusing on identifying high-demand skills, salary trends, and hiring patterns across major cities.

## Key Insights
- The most in-demand technical skills such as *Data Cleaning, SQL, Excel, and Python*
- Variations in average salaries across cities, with certain regions showing higher compensation levels
- Differences in salary distribution across data-related roles (e.g., Data Scientist, BI Analyst, Data Analyst)
- Job demand concentration in major hiring hubs like *Riyadh*

## Project Overview
The project is designed as a business-focused dashboard to simulate real-world analytical reporting and support decision-making.
