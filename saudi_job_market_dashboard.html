Saudi Job Market Analysis Dashboard

Portfolio Project | Data Analysis • Visualization • Business Insights
This project explores a realistic demo dataset of job postings in Saudi Arabia to identify in-demand skills, salary trends, and hiring patterns across major cities. It is designed as a portfolio-ready business-facing analysis project for a Data Analyst profile.

Key Insights

  • Data Cleaning and Data Visualization are among the most frequently mentioned skills across roles.
  • Khobar shows the highest average salary in the dataset, while Riyadh has the strongest hiring demand.
  • Data Scientist roles command the highest average salary among analyzed job titles.
  • Dashboard and visualization tools remain highly relevant alongside SQL and Python.

Recommendation: Candidates targeting Data Analyst roles in Saudi Arabia should focus on SQL, Python, and dashboarding skills, especially for opportunities in Riyadh and Jeddah.
